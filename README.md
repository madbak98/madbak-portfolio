# Madbak Portfolio

A creative portfolio site built with Next.js: scroll-driven interactions, a Three.js background scene, multilingual content, and a responsive layout.

## Tech stack

- **Next.js** (App Router)
- **React**
- **TypeScript**
- **Tailwind CSS**

## Features

- Animated UI and scroll-based interactions
- Portfolio showcase with project detail views
- Responsive design
- Internationalization (EN / FA / TR)

## Prerequisites

- Node.js 18+ (recommended: current LTS)

## Installation

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm run start
```

## Environment variables

Copy `.env.example` to `.env.local` and set values as needed. Social link overrides are optional; see `app/lib/portfolio-data.ts` for defaults.
